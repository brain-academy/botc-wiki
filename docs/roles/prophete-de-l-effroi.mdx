### Description

Chaque nuit, choisissez un joueur. Si vous le nominez et qu’il est exécuté, son équipe perd. Tous les joueurs savent quand vous choisissez un nouveau joueur.

Le prophète de l’effroi créé de la paranoïa sur qui nomine qui.

-   Durant la première nuit, quand le prophète de l’effroi choisit un joueur, tous les joueurs l’apprennent.

-   Durant les autres nuits, chaque fois que le prophète de l’effroi choisit un nouveau joueur, tous les joueurs l’apprennent. Si le prophète de l’effroi sélectionne le même joueur que précédemment, les joueurs n’apprennent rien.

-   Les joueurs apprennent seulement que le prophète de l’effroi a agi. Ils n’apprennent pas quel joueur a été sélectionné.

-   Si le prophète de l’effroi nomine le joueur qu’il a choisi et que cette nomination se termine en exécution, le joueur choisi perd avec son équipe et la partie est terminée.

-   Seul le joueur actuellement choisi est affecté par la capacité du prophète de l’effroi. Les joueurs choisis précédemment ne comptent pas.

-   Si le joueur choisi est exécuté mais ne meurt pas, le joueur choisi et son équipe perdent quand même.

### Exemples

> La première nuit, le prophète de l’effroi choisit le majordome. Tous les joueurs savent que le prophète de l’effroi a choisi. Durant la journée, le prophète de l’effroi nomine le majordome qui est exécuté. Les maléfiques gagnent.

> La nuit, le prophète de l’effroi choisi le jongleur. Le fleuriste nomine le jongleur et le jongleur est exécuté. La partie continue.

> Le prophète de l’effroi choisit l’empathique. La nuit suivante, le prophète de l’effroi choisit le soldat. Le prophète de l’effroi nomine et exécute l’empathique. La partie continue car le prophète de l’effroi a sélectionné le soldat et non l’empathique.

> Le prophète de l’effroi choisit accidentellement le baron à cause de la capacité du cultivateur de pavot qui est jeu. Le prophète de l’effroi nomine et exécuté le baron. Le village gagne.

### Comment conter

Chaque nuit, réveillez le prophète de l’effroi. Il désigne n’importe quel joueur. Rendormez le prophète de l’effroi. Marquez le joueur choisi avec le jeton de rappel « Effrayé ». Si le prophète de l’effroi choisi un joueur qui n’était pas déjà marqué « Effrayé », déclarez que « le prophète de l’effroi a choisi un joueur. » (_Cela informe le village qu’un prophète de l’effroi est en vie et qu’il a choisi un nouveau joueur._)

Si le prophète de l’effroi nomine le joueur marqué « Effrayé » et que la nomination entraîne l’exécution, déclarez que la partie est terminée et quelle équipe a gagnée.
