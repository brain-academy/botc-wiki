### Description

Si le démon tue le roi, vous apprenez qui est le démon. [+ le roi]

L’enfant de chœur apprend qui est le démon quand le roi est assassiné.

-   Le roi peut être en jeu sans l’enfant de chœur. Durant la phase d’installation, si l’enfant de chœur est en jeu et que le roi ne l’est pas, le roi est ajouté. Si le roi est déjà en jeu, l’enfant de chœur n’ajoute pas un deuxième roi.

-   Si le démon tue le roi en utilisant sa capacité, l’enfant de chœur apprend qui est le démon. Le démon qui nomine et exécute le roi ne compte pas. Les sbires qui tuent le roi, comme l’assassin, ne comptent pas non plus. 

-   Si le démon attaque le roi mais que celui-ci ne meurt pas, l’enfant de chœur n’apprend pas qui est le démon. 

-   L’enfant de chœur apprend quel joueur est le démon mais pas le rôle du démon.

### Exemples

> Le <Role IMP /> attaque l’<Role EMPATHIQUE />. L’<Role EMPATHIQUE /> meurt. La nuit suivante, le <Role IMP /> attaque le <Role ROI /> qui est protégé par le <Role MOINE />. Le <Role ROI /> reste en vie. La nuit suivante, le <Role IMP /> attaque le <Role ROI /> qui n’est plus protégé par le <Role MOINE />. Le <Role ROI /> meurt. L’<Role ENFANT_DE_CHOEUR /> est réveillé par le conteur pour apprendre quel joueur est le <Role IMP />.

> Le <Role SHABALOTH /> tue le roi. L’<Role ENFANT_DE_CHOEUR /> ivre se réveille et apprend faussement que le joueur, qui est en réalité le <Role GENERAL />, est le démon.

### Comment conter

En préparant la partie, avant de mettre les jetons de rôles dans le sac, si le roi n’est pas déjà en jeu, retirez un jeton de rôle de villageois et mettez le jeton de rôle de roi à la place.

Chaque nuit, sauf la première, si le démon tue le roi, rendormez le démon et réveillez l’enfant de chœur. Désignez le joueur qui est le démon puis rendormez l’enfant de chœur.
