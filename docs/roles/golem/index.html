Golem

Modules : Expérimental

Nom Original : Golem

Alignement : Etranger - Gentil/Bon

Classe : Spécial

Réveil : Jamais

Description​

Vous ne pouvez nominer qu’une fois par partie. Quand vous le faites, si le nominé n’est pas le démon, il meurt.

Le golem tue le joueur qu’il nomine.

  • Quand le golem nomine un joueur, ce joueur meurt immédiatement. La nomination continue.

  • Si le golem nomine le démon, rien ne se passe. Le conteur ne confirme pas et ne nie pas que le golem a nominé et continue avec le processus de nomination. Le conteur pourrait dire « rien ne s’est passé. » si une clarification est demandée.

  • Après que le golem ait nominé une fois, que le nominé soit mort ou non, le golem ne peut plus nominer sur cette partie. C’est la responsabilité du joueur de se retenir de nominer, pas celle du conteur. Nominer délibérément alors qu’on ne le doit pas est considéré comme de la triche.

Exemples​

Le Golem nomine le Cultivateur de pavot. Le Cultivateur de pavot meurt. Le Golem ne peut plus nominer de la partie.

Le Golem nomine le Reclus. Le Reclus est alors considéré comme démon. Rien ne se passe et le conteur commence à compter les votes pour l’exécution du Reclus. Le Golem ne peut plus nominer de la partie.

Comment conter​

Si le golem nomine un joueur qui n’est pas démon, ce joueur meurt, puis le vote continue comme d’habitude. Marquez le golem avec le jeton de rappel « ne peut pas nominer ».

Si le golem nomine le démon, le vote continue normalement (Ne dites pas pourquoi.) Marquez le golem avec le jeton de rappel « ne peut pas nominer ».

Si le golem nomine accidentellement alors qu’il ne le devrait vous pouvez accepter ou refuser cette nomination. Il est préférable de ne pas accepter la nomination tant que vous pensez que ça ne confirme pas que le golem est en jeu. Pour rester juste, faites aussi cela pour les joueurs qui bluff golem en leur rappelant qu’un golem ne peut pas nominer plus d’une fois.

Compléments conteur

Gardez en tête que le joueur nominé par le golem n’est pas exécuté. Il meurt simplement. Cette capacité ne surpasse pas les capacités protégeant de la mort.

Le golem peut se nominer et se tuer lui-même indiquant ainsi être le golem même si dans ce cas, on ne peut pas le distinguer d’une mort de sorcière.

Si le golem est ivre ou empoisonné, le nominé ne meurt pas quel que soit son type mais la nomination du golem est toujours utilisée. Techniquement, le golem peut nominer une deuxième fois s’il est ivre ou empoisonné un jour après avoir utilisé sa première nomination. Toutefois, comme pour le majordome, le golem ne peut pas nominer délibérément une deuxième fois pour essayer de tester s’il est ivre ou empoisonné après avoir utilisé sa nomination unique de la partie. S’il le fait, vous pouvez le permettre comme l’interdire. Si le golem a tué lors de sa première nomination et qu’il s’est ainsi confirmé, mieux vaut l’interdire car vous ne confirmez rien que le village ne sait pas déjà (avec quelquefois où vous le permettez afin de semer le doute sur le fait qu’il ait été ivre/empoisonné ou pas.) Si le golem n’a pas tué lors de sa première nomination, vous pouvez le permettre car ça le fait paraître juste plus maléfique.
