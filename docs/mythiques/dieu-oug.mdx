### Description

Un chapeau Oug. Quand porter chapeau Oug, on doit parler un son à la fois mais voter deux fois. Si échec, passe chapeau Oug.

Le Dieu Oug fait parler les joueurs avec des mots monosyllabiques.

-   Quand le joueur portant le chapeau Oug vote, son vote compte double. Utilisez un chapeau physique si vous le pouvez.

-   Tous les mots qui ont une syllabe comptent. Les contournements à base de mots-valises ne comptent pas. Prononcer un seul mot multisyllabique est suffisant pour perdre le chapeau Oug.

-   Le Conteur choisit qui porte le chapeau Oug en premier. Si un joueur prononce un mot multisyllabique ou si le joueur prend trop de temps à parler, ou si le joueur ne s’amuse pas, ou si le joueur a le chapeau depuis trop longtemps, le Conteur choisit un nouveau joueur pour porter le chapeau.

-   N’importe quel joueur peut porter le chapeau Oug. Il peut le refuser s’il le souhaite. Un joueur peut avoir le chapeau Oug plus d’une fois.

-   Si le Conteur n’est pas présent, on se repose sur le code d’honneur. N’importe quel joueur qui surprend le joueur portant le chapeau Oug prononcer un mot multisyllabique peut le dénoncer et le chapeau Oug sera alors retiré au joueur.

-   Le Conteur peut choisir un nouveau joueur chaque jour pour porte le chapeau Oug ou garder le même joueur qu’au jour précédent. 

### Exemples

> Maëlle porte le chapeau Oug. Une fois nominée, elle dit « C’est pas moi. Je crois que Lex a bla avec gars mal. Il est mal. » Maëlle garde le chapeau.

> Zariko a le chapeau Oug. Dans une conversation privée avec Mathis et Tiphaine, Zariko dit « Je crois que vous est tous les deux bons. Je suis le piaf qui peut voir un gars quand il meurt. Je veux que le Démon me tue pour que je prends ce gars et vous dire truc sur lui jour plus tard. » Le Conteur donne le chapeau Oug à Florian parce que le mot « Démon » est un mot multisyllabique.

### Comment conter ?

Quand la partie commence, donnez le chapeau Oug à n’importe quel joueur. Si vous ou n’importe quel joueur surprenez ce joueur prononçant un mot multisyllabique, donner le chapeau Oug à un autre joueur.

Si le joueur portant le chapeau Oug vote, son vote compte comme deux votes.

_Règle optionnelle_ : Si porter le chapeau Oug n’est plus drôle, limitez-le aux phase de nominations uniquement._
